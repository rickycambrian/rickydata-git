# Research Notes

Research in this directory records design input for the local-only foundation
phase. The current implementation intentionally remains read-only with respect
to Git metadata.

Primary references:

- Git object/ref compatibility
- gitoxide/gix Rust Git plumbing
- Radicle-style repo-native collaboration objects
- Jujutsu-style operation and workspace isolation concepts
- Rickydata sign-to-derive and TEE receipt patterns
- RDL manifests, traces, capabilities, and fix safety
